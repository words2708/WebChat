# WebChat 
Firebase based webchat using vanilla javascript

Features:
1. 2 User Login based on emails (by clicking "User1 Login" button or "User2 Login" button) provided within code (user: words2708@gmail.com, passwd: pass123, user2: srividhya.sub@gmail.com, password123) to view chat history and new messages as they arrive.

2. Multiple instances of the webchat app can help exhibit multiple user logins of app showing real time messages as they arrive 

3. One can set a simple blue wallpaper for look 'n' feel sakes, by clicking the "Set Wallpaper" button.

4. Long messages can wrap horizontally automatically & scroll vertically using down arrow or with touch/mouse from within the chat message bubble.
